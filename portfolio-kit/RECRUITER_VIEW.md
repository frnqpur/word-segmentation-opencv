# Recruiter View — Word Segmentation using OpenCV

## One-Line Summary
An OpenCV-based computer vision project that segments handwritten text regions using image preprocessing, contour detection, bounding boxes, and Streamlit visualization.

## What This Project Shows
This project demonstrates the ability to:

- Work with computer vision workflows.
- Process image data using Python and OpenCV.
- Apply grayscale conversion and thresholding.
- Use morphological operations for text-region grouping.
- Detect contours and draw bounding boxes.
- Build a simple demo interface using Streamlit.
- Present a technical project in a recruiter-friendly portfolio format.
