# CV Bullets — Word Segmentation using OpenCV

## English Version

### Short Version
- Developed an OpenCV-based computer vision pipeline for handwritten text-region segmentation using grayscale conversion, binary thresholding, morphological dilation, contour detection, and bounding-box visualization.

### Detailed Version
- Built a Python and OpenCV image processing workflow to segment handwritten text regions from images, including preprocessing, thresholding, dilation, contour detection, bounding-box sorting, and segmented crop extraction.

### Portfolio-Oriented Version
- Created an interactive Streamlit demo for an OpenCV-based word segmentation project, allowing users to upload handwritten images, view preprocessing results, inspect bounding boxes, and review segmented word/component crops.

### Technical Version
- Implemented classical computer vision techniques using OpenCV and NumPy to process handwritten images, detect contours, filter text-like regions, and generate bounding-box visualizations for segmentation analysis.
